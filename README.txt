Execution:
1. Install Python
2. Run "game.py" on terminal

Introduction:
Turn-Based mini-game.
Player have 3 monsters, enemy has 1.
Player randomly pick one monster and fight with enemy until it died and pick another one randomly again.
Win condition - enemy died
Lose condition - all player monster died
You can change the value of monster yourself
Line 42-44: Player's 3 monsters
Line 46: Enemy monster
The following values are (name,hp,atk,shield)

執行:
1. 安裝 Python
2. 在terminal 執行指令 "game.py"

介紹:
回合制小遊戲
玩家有三隻魔物，敵方有一隻魔物
玩家隨機抽出一隻魔物與敵方魔物戰鬥直到戰死並抽出下一隻
勝利條件 - 敵方魔物死亡
失敗條件 - 玩家魔物全數死亡
魔物數值玩家可以自行更改 
Line 42-44 為玩家三隻魔物的數值
Line 46 為敵方魔物的數值
順序為(名字,血量,攻擊力,護盾)
